¿Por qué usar AngularJs?

  • Web dinámicas
    • Porque ayuda con la declaración de vistas dinámicas en las aplicaciones web, algo que resulta complicado de realizar solo con HTML
  • Orden y comprensión
    • Porque extiende el vocabulario de HTML y así es posible desarrollar de mánera rápida aplicaciones ordenas y entendibles
  • Rapidez
    • Porque crea Web rápidas que no gastan recursos cargando páginas completas
  • Fácil de "testear"
    • Porque permite realizar pruebas unitarias o de "extremo a extremo" (Mocks)
